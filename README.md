# BFHL API – Qualifier 1

This project implements the required REST APIs for Bajaj Finserv Qualifier 1.

## APIs Implemented

### POST /bfhl
Handles the following inputs:
- `fibonacci` → returns Fibonacci series
- `prime` → returns prime numbers from array
- `lcm` → returns LCM of array
- `hcf` → returns HCF of array
- `AI` → returns a single-word response using Google Gemini API

### GET /health
Health check endpoint.

## Tech Stack
- Node.js
- Express.js
- Google Gemini API
